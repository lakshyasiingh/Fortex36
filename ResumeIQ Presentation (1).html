ResumeIQ

AI-Powered Resume Checker & ATS Scanner

Intelligent resume optimization and ATS compatibility analysis powered by AI

Problem Domain: Career Development & Recruitment Technology

GitHub: github.com/lakshyasiingh/Fortex36

Problem & Motivation

  • Problem: Job seekers spend hours on resumes without knowing if they'll pass ATS filters or catch recruiter attention
  • Gap: Existing tools lack real-time AI feedback on resume quality, keyword optimization, and ATS compatibility
  • Impact: Qualified candidates lose opportunities due to poor resume formatting and keyword mismatch
  • Solution Need: Automated, intelligent resume analysis that provides actionable, personalized feedback
GitHub: github.com/lakshyasiingh/Fortex36

Solution Overview

Core Features

  • ATS Compatibility Scanner: Analyzes resume format, structure, and keywords against ATS algorithms
  • AI-Powered Feedback: Identifies strengths, weaknesses, and actionable improvements
  • Keyword Optimization: Suggests industry-specific keywords and skills to match job descriptions
  • Real-time Score: Generates ATS compatibility score and readability metrics
GitHub: github.com/lakshyasiingh/Fortex36

Technology & Architecture

Frontend

React.js, Next.js, Tailwind CSS for responsive UI

Backend

Node.js, Express.js, REST APIs for resume processing

AI/ML

NLP models (spaCy, BERT) for text analysis and keyword extraction

Database

MongoDB for resume data storage and user profiles
